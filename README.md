# LoraBoard
A custom ESP32-S3 LoRa development board designed for long-range wireless communication projects.

## Overview
LoraBoard is a custom PCB designed around the ESP32-S3 microcontroller and the SX1262 LoRa transceiver. The board is intended for low-power, long-range communication applications such as mesh networking, remote monitoring, IoT devices, and sensor networks.

The project was designed using KiCad and includes a complete schematic, PCB layout, and manufacturing files.

## Features
- ESP32-S3 microcontroller
- SX1262 LoRa transceiver
- USB connectivity for programming and power
- Compact PCB design
- Long-range wireless communication
- Suitable for Meshtastic and IoT applications
- Hardware
- Main Components
- ESP32-S3
- SX1262 LoRa Radio
- USB Interface
- Antenna Connector
- Repository Structure
